#Pokemon Snap
===========================
Spencer King, Hannah Porter, David Wu
CSE20212 Final Project, Spring 2014

#How to Install
Libraries: SOIL, libjpeg, FMOD Ex

#Controls
WASD - lateral movement
Up/Down - vertical movement
E - snap picture
M - pause music
Mouse - perspective
Q - quit

#What we learned
OpenGL, makefiles, linking libraries
